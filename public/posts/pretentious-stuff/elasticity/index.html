About this project

This is a simple, ‘from-scratch’ and (maybe) hardcoded FE script, nonetheless it gets the job done. This is one of the first scripts I did for the Theoretical and Computational Mechanics Group. There is not much to say, but that the code is thought for both planar stress or planar strain. The following example includes a one sided fixed plate pulled by a opposite side distributed load. The tackled problem is shown in the following figure.
